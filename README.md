ApexLabs is a responsive e-commerce website for selling gaming PCs and components.
It showcases products dynamically loaded from JSON files and provides users with a smooth intuitive shopping experience across all devices.

## Features
### Pages:
- **Home:** Hero section with featured products, why choose us section, and fade transitions
- **Shop:** Browse 4 product categories (PCs, Graphics Cards, Motherboards, CPUs) with price sorting
- **Contact:** Contact form with validation, success/error messages, and business info sidebar
### Other Features:
- Fully Responsive Design
- Product loading from JSON files
- JS Form Validation
- Hamburger menu navigation for smaller screens
- Smooth animation & hover effects
- Accessible (ARIA labels for screen readers)
### Disclaimer:
- Some footer links are invalid and do not link to anything. (About Section, Support Section)

## Installation Guide (Run Locally)
### Prerequisites:
- Git
- VSCode (Open Live Server extension)


### Setup (locally):
1. **Clone the repository:**
   ```bash
   git clone https://github.com/TommyNam/Apex-Labs.git
2. **Navigate and Select the correct root directory (Important for correct routing)**
   ```bash
   File -> Open Folder -> Apex-Labs
4. **Install VSCode's 'Open Live Server' extension**
5. **Inside VSCode, select /public/index.html**
6. **Right click 'index.html' and click 'Open with Live Server'**
